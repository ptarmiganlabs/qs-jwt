qs-jwt — Command Line Tool for Creating Qlik Sense JWTs
=========================================================

Version: __VERSION__

Getting Started
---------------

Run the binary from a terminal or command prompt:

  ./qs-jwt --help        (Linux/macOS)
  qs-jwt.exe --help      (Windows)

For Qlik Sense Cloud:
  qs-jwt create-qscloud --help

For Qlik Sense Enterprise on Windows (client-managed):
  qs-jwt create-qseow --help

Full documentation:
  https://qs-jwt.ptarmiganlabs.com

A cross-platform command line tool for creating JWTs (JSON Web Tokens) that can be used to authenticate with Qlik Sense Enterprise on Windows (QSEoW) and Qlik Sense Cloud.

**Features:**

- Cross-platform: Windows, Linux, macOS
- Create JWTs for both QSEoW and Qlik Sense Cloud
- Use existing certificates or create new ones
- Perfect for automation and CI/CD pipelines
- Rich JWT claims support for security rules

qs-jwt nicely complements the other open source tools in the [Butler family](https://github.com/ptarmiganlabs).
